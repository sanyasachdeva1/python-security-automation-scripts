# Sample Security Automation Report

## Summary

This report shows sample findings generated by the Python security automation scripts.

## Finding 1: Brute Force Login Activity

Severity: High

Suspicious IP addresses were detected with repeated failed SSH login attempts.

Affected IPs:

- 45.33.21.10
- 103.25.44.8

Recommendation:

- Block suspicious IPs at firewall or security group level
- Enforce MFA
- Disable password-based SSH login
- Use key-based authentication

## Finding 2: IOC Match

Severity: High

A known suspicious IP address was found in authentication logs.

Matched IOC:

- 45.33.21.10

Recommendation:

- Investigate the source IP
- Search SIEM logs for related activity
- Check whether any successful login happened after failed attempts

## Finding 3: Exposed Services

Severity: Medium

Nmap scan identified open services.

Open ports:

- TCP/22 SSH
- TCP/80 HTTP

Recommendation:

- Restrict SSH access
- Review firewall and security group rules
- Disable unnecessary exposed services

## Finding 4: Risky IAM Policy

Severity: High

The IAM policy contains risky permissions.

Risky permissions:

- Action: *
- Resource: *
- iam:PassRole

Recommendation:

- Remove wildcard permissions
- Apply least privilege
- Scope resources to specific ARNs
- Review privilege escalation paths
